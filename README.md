# sb-react-lection
React

For starting work:
1) Create folder for Application like 'test-aplication'
2) Go to this directory
3) git clone git@github.com:AlexandrBursak/sb-react-lection.git .
4) run command:
$ npm update
5) start live server
$ npm run start
6) go to browser and load link:
http://127.0.0.1:1234

For updating projects:
$ npm run dev
